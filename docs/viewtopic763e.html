Hello Chen,

There are the nice PAW pseudos - 12 valence electrons - from Bernard Amadon and used for his Physical Review B 77, 155104 (2008) paper. They are accessible at http://www.abinit.org/downloads/PAW/MAI ... ATA/058-Ce
And if you want to include more valence, the provided Atompaw input files are a good start to build yours. Run the generator once interactively to input the parameters by hand and you'll find quite easy to modify them. Don't forget to adjust Rpaw.

Kind regards,

Alain
