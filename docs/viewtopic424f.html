You can always use the UNK in binary form:

First, run ABINIT to produce the UNK binary files.

Then run Wannier90 in stand alone mode to read the binary UNK files and to produce the .xsf files.
Just keep : wvfn_formatted=.false. in the .win file,
You need to add the input variables in w90.win to plot the Wannier functions in xsf format.

In other words, Wannier90 can read the UNK files in binary format to produce .xsf files.
