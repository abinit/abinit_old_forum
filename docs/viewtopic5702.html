Dear Ziye,

I think that "--enable-netcdf" is not a known option

try to use :

Code: Select all

--with-trio-flavor="netcdf"


at the end of configure output, you must see in the "Final Remarks" :

Code: Select all

* TRIO   flavor = netcdf
