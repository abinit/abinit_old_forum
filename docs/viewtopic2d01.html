New Release of the NanoTCAD ViDES code

Posted: Fri Aug 30, 2013 10:42 am
by fiori
We are pleased to announce that a new version of the open-source NanoTCAD ViDES code has been released and available for download at http://vides.nanotcad.com

The code is now able to interface with Wannier90 (and Abinit through Wannier90/Abinit interface) in order to perform multi-scale simulations of nanoscale devices. Look at the tutorial on multi-scale simulations of functionalized graphene nanoribbon Field Effect Transistors for an understanding of its capabilities (http://vides.nanotcad.com/vides/documen ... simulation).

The code efficiently computes transport in generic structures defined at run time by the user and it is particularly geared towards the performance assessment of carbon/graphene based devices through the self-consistent solution of Poisson and transport equations.

Enjoy!
