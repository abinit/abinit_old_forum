Dear Steve,

In the module src/68_dmft/m_green.F90, you have the subroutine print_green. It
this subroutine, you have some lines devoted to the writing of spectral functions (only for the Hubbard I solver).
To have the spin resolved spectral function, you should modify this section slightly.

Bernard
