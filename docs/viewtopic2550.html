GUI is a user friendly interface to systems. Unlike a command line operating system or CUI like Unix or MS-DOS, GUI Operating Systems are much easier for end-users to learn and use because commands do not need to be known or memorized. Because of their ease of use, GUI Operating Systems have become the dominant operating system used by end-users today. So how much friendly it is, the demand increases because of ease of use.
