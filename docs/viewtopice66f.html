Hello Joe,

each object in Plone has a "Title" and a "Short name"

the Title is set by the Marc's procedure but the "short name" is created automaticaly by the Plone
You can set the "Short name" by renaming the folder :

1) go to the content of ATOMICDATA folder :

http://www.abinit.org/downloads/PAW/CON ... r_contents

2) select the folder and clic on the "Rename" bouton
3) set the "New short name"
4) Save

I have made this for the Ba...
Try with Nb...
