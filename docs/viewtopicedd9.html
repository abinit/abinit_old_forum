Dear quantumNoob,
The atlas library is a mathematical library (blas and lapack) but you can install them from any other source (gnu, mkl, etc). If you want to go simple (but this is not the most efficient for running production calculations) is to simply do "sudo aptget install libblas-dev liblapack-dev" and they can be linked by adding with_linalg_libs="-L/usr/lib/ -llapack -lblas" in the .ac file (check they are installed in /usr/lib).
For netcdf, libxc, etc, if you just declare you want to install them in your .ac file but do not give a link to them (with_trio_flavor="netcdf") (with_dft_flavor="libxc" for example), Abinit will try to download them and install them automatically. This is strongly recommended to be sure the installed version of these fallbacks are in agreement with your abinit version.
Best wishes,
Eric
