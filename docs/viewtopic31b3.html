New GUI in abinit v6.8

Posted: Fri Apr 22, 2011 1:51 pm
by mverstra
Hello everyone,

this is more of an announcement: a new GUI coded in java will be distributed as a beta version with abinit 6.8 (inside the package itself), which should appear in the coming month or so. It is very featureful, with remote access to clusters and copying of files. It was made by Flavio Abreuaraujo, a Master's student in the group at UCLouvain.

This new sub-forum is for all questions regarding the GUI, but also other gui-related stuff (e.g. a future version of Avogadro will be abinit enabled)

Cheers

Matthieu
