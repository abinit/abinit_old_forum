ABINIT v8.4.3 Hotfixes

Posted: Mon Aug 14, 2017 8:16 am
by gonze
Dear ABINIT users,

ABINITv8.4.3 is available, see the Web page http://www.abinit.org/downloads/abinit-packages .

*** This release of ABINIT is a production version ***

The following fixes have been made with respect to v8.4.2, among others :

when nspinor=2 and nspden=1 (so, non-magnetic system, but with spin-orbit), fix LDA+U as well as local exchange
modification to avoid internal compiler error with intel2013_sp1.1.106
improvements and fixes for the reading and initialization of the density, especially in the nspden=4 case (non-collinear magnetism)
fix gwfockmix support
Have fun !

Xavier
