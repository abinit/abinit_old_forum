Thank you for your reply, Eric - apologies for not being clearer.

I have resolved this issue. My problem was that I was not using the right environment variables when building the library (no plasma.mod file was created), then linking the static libraries in the correct order.

For anyone interested, PLASMA 2.8.0 is compatible with ABINIT 8.10.3, although a modification to the src file "src/28_numeric_noabirule/m_abi_linalg.F90" (see viewtopic.php?f=3&t=3166&p=9767&hilit=PLASMA#p9767 )
