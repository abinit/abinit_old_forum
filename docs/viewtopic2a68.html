Post by Jordan » Fri Mar 13, 2015 9:47 am

Please use enable_debug="enhanced" in your *.ac file to configure and compile abinit, or alternativaly (assuming you use gfortran compiler) set FCFLAGS_EXTRA="-fbacktrace -g -O0" in the same *.ac file.

We cannot help if we don't know what file and what line is problematic.

Cheers

First, note that Abinit does not support intel15 yet. We are still working on it. Please use intel14 or gfortran4.9.x if you want the most recent compiler.

Although Optimization is set to "no" you do have optimizations. ("-O3 -ip") which might trigger the problem with intel15.

Please set in your *.ac file
FCFLAGS="-O0 -g -traceback -static-intel" # if you use intel14 compiler
FCFLAGS="O0 -g -ftraceback" # if you use gfortran.

You can try to add "-check all" for intel14 or "-fcheck=all" for gfortran to the previsous definition but it will considerably slow down the execution.
