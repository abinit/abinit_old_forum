Just to clear up some issues that I had. I believe I found some of the answers I was looking for.
xo_SIG contains Re{Sigma}, Im{Sigma}, and the spectral function for each k-point, band, and energy value (from freqsp). The file is set up in the following format:
# k = X
# band from Y to Z
freq-point ... Re (band Y) ... Im (bandY) ... spectral function (bandY) ... up to band Z
