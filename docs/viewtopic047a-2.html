the input files for the tutorials are in the abinit distribution in ~abinit/tests/tutorials/Input/ and ~abinit/tests/tutorespfn/Input/

Also, as usual, check the other test files (README contains the descriptions) for the themes and applications you are interested in - this points you to tests which are pertinent for you and your input variables.

No one will make your input for you! The documentation is ample enough to get you through if you search a bit (grep and google).
