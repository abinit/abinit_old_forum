Hello,

1) if this is for gamma, you have quite a large acoustic sum rule break. You should run anaddb with asr 1 or 2, and check your pseudopotential and ecut convergence.

2) you should start with the theory, e.g. read chap 27 of Ashcroft and Mermin. The LO mode is the non-degenerate, higher frequency mode.

3) to calculate the LO-TO splitting, follow the response function tutorials for AlAs carefully, and use qph2l / nph2l in anaddb
