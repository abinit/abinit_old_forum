You are trying to read a WFK file generated with a recent version of Abinit but the optic
tool found in your $PATH is too old and does not support the new fileformat.

Use the shell command:

$ whereis optic

to print the location of the executable.
Make sure you are using a version of optic that matches the one used to generate the WFK file

Re: optical erreur

Posted: Wed Apr 02, 2014 5:02 pm
by gmatteo
Note that I was suggesting to change the version of optic, and not the version of Abinit

The version of optic you are using is too old.
OPTIC-5.3.4 cannot read the WFK file produced by Abinit-6.10.3
since we have introduced new formats for the WFK file.
Use Optic version 6.10.3, the executable is located inside ~abinit-6-10.3/src/98_main/
where ~abinit is the directory containing the source code used to compile the abinit executable
you are using in your calculations.
