Hello,

This forum is supposed to be international. You should consider writing in english preferably, you will get much more answers from other people. For the time being, I would recommend you to make the tutorials of ABINIT in order to understand some part of the input files at least. I think that nobody will spend time to make the whole input file for you. First make the tutorials 1, 2, 3 and 4, and read carefully the associated input variables (basic structure variables, such as xcart, rprim, acell, gs variables such as ecut, ngkpt, ... etc) then try to make an input file for your wurtzite system. Then if you have more specific problems, you could come back to this forum.

Good luck
