by ebousquet
Dear Markus,
At present the only non-linear properties (3rd orders) that can be computed directly with DFPT are the electro-optic coefficients and raman tensors (meaning with intensities).
For the other cases, you should combine 2nd order DFPT and frozen phonon (direct method) techniques (for electrostriction, etc).
All the best,
Eric
