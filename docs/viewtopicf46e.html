Dear ABINIT users,

ABINIT version 8.2.2 is available ...
You will find it on the Web site, starting from
http://www.abinit.org/downloads/abinit-packages

*** This release of ABINIT is a production version ***

In ABINIT v8.2.1 tar.gz, some files were lacking in order for the automatic tests to run. This
has been fixed in ABINITv8.2.2 .
Also some documentation files have been upgraded, to take into account to the replacement of bzr by git.

Have fun !
