Hello Sana,

You probably don't have enough credentials on your Suse box to write to /usr/local ... this directory is owned by administrators and "make install" should be invoked as root to be allowed to dump files in that place. So either use a "sudo make install" (enter your login password and pray to be in the sudoers list) or "su root" (if you know the root password) and then a "make install" ... or you would have been wiser to configure the compilation with the "--prefix=..." option and specify a path somewhere in your home directory.
