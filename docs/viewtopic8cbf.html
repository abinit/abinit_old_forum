Bugs in optics : fixed in v7.4.2  [SOLVED]

Posted: Sat Aug 17, 2013 10:23 am
by gonze
Dear ABINIT users,

After the v7.4.1 has been frozen (you can download it from the Web site),
two bugs have been discovered and fixed in optics. They happen only
when symmetries are used to complete the set of k points (kptopt/=3). Such problem was
also present in all versions of optics, before v7.4. The fixes will be in v7.4.2 ,
available mid-September.
