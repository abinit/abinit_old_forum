You can't optimize the conventional cell but not the primitive cell, they're different representations of the same thing. It's always preferable to work with the primitive cell because it's so much smaller. And, abinit will give the same answers on Windows as on Linux (it's just a different representation of the same thing posting.php?mode=reply&f=8&t=2056#
