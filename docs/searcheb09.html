Hello,

When doing GW calculations with abinit, you have to specify the k-point mesh with ngkpt or kptrlatt. So the solution would be to just do like this :

kptopt 1
ngkpt 1 1 1
nshiftk 1
shiftk 0.0 0.0 0.0

Good work

Re: how to specify the typat when the number of atoms is big

Hello,

You can use the star (*) in this way :

Instead of typing

typat 1 1 1 1 1 2 2 2

You can type

typat 5*1 3*2

meaning that you have 5 atoms of type 1 and 3 atoms of type 2.
Of course you have to group the atoms of the same type to use this trick...
