Dear Vladimir,
I think what you want is controlled by the prtvol variable:
http://www.abinit.org/doc/helpfiles/for-v8.0/input_variables/html_automatically_generated/varfil.html#prtvol
By default prtvol=1 will print only the first 50 k-points, switch to prtvol=2 to get all of them.
Best wishes,
Eric
