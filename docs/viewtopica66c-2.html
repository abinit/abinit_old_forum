Hi, you need to use either rprim or angdeg.

from :http://www.abinit.org/documentation/helpfiles/for-v6.2/input_variables/varbas.html#angdeg
Gives the angles between directions of primitive vectors of the unit cell (in degrees), as an alternative to the input array rprim . Will be used to set up rprim, that, together with the array acell, will be used to define the primitive vectors.


In case you input angdeg, don't bother about computing rprim. Abinit will perform this calculation for you. Just keep in mind to tell abinit about acell.
