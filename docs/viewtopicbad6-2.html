Tests v5/t112 and v5/t113 got moved to the tests/seq group after I re-wrote the PAW Berry's phase implementation code. The original implementation only worked in serial, I added various changes that included parallelization over k points. In v5, no version of t112 or t113 is actually executed (see v5/tests.cnf, those entries are among the ones commented out).
