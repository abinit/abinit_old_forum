Re: Problem in parallelizing abinit 6.4.3

by Alain_Jacques
Hello Renyu,

I cannot say that Abinit is well tested against LAM. Unfortunately, this MPI variant has no Fortran 90 wrapper i.e. you don't have a /homes/renyu/pkgs/lam-7.1.4/bin/mpif90, don't you?
A possible workaround is to use the mpif77 wrapper and tell it to behave properly. Try

Code: Select all

../configure --enable-mpi="yes" FC="/homes/renyu/pkgs/lam-7.1.4/bin/mpif77 -f77=%none -ftrap=common " CC="/homes/renyu/pkgs/lam-7.1.4/bin/mpicc" CXX="/homes/renyu/pkgs/lam-7.1.4/bin/mpic++"
Add other options you need but no --mpi-prefix.
Working?

Kind regards,

Alain
