Hi,

I believe there is some bug with the setup of the fft meshes for large cells.
Here is a work-around. Look in the log for a sensible fft mesh to use. Then can set the fft mesh manually with e.g.

fftgw 00
ngfft 24 24 180
All times are UTC+02:00
