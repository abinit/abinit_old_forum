ABINITv9.4.2 production release.

Posted: Sat May 01, 2021 5:14 pm
by gonze
Dear ABINIT users,
ABINITv9.4.2 is available, see the Web page https://www.abinit.org/packages

*** This release of ABINIT is a production version ***

Compared to ABINITv9.4.1, there have been several minor bug fixes, one of which was related to a significant CPU time waste when initializing a run from an existing density/potential file if the FFT grid was different and many processors were used. Several fixes of documentation have been made also.

As a reminder, going from v9.2 to v9.4. all the tutorials have been carefully reexamined and improved.

More modifications are mentioned in the v9.4 release notes https://docs.abinit.org/about/release-notes.

Have fun !

Xavier
