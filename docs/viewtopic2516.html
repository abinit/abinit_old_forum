Post by gabriel.antonius » Thu May 09, 2013 3:26 pm

The problem is
getwfk?2 1

Since you use multi dataset mode with
ndtset 18
udtset 9 2
the first dataset index is "11" and the second is "12", but now it's looking for dataset "1"
Just use
getwfk?2 -1
