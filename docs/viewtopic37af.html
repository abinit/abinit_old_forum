ABINIT v8.4.4 Hotfixes and more ...

Posted: Wed Sep 20, 2017 1:54 pm
by gonze
Dear ABINIT users,

ABINITv8.4.4 is available, see the Web page https://www.abinit.org/package/8.4.4

*** This release of ABINIT is a production version ***

This version goes a bit beyond fixes, as the support of Libxc 3.0 has been activated (Libxc 2.2 can still be used),
support of ELPA 2016 and 2017 activated, and the version of Atompaw upgraded.
Fixes to hyperlinks, to accomodate the new ABINIT Web site have been made.
Other bug fixes include fixing problems related to getcell & getwfk and electron-positron calculations. Special thanks to M. Torrent
for these modifications with respect to v8.4.3.
Have fun !

Xavier
