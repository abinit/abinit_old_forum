Usually the test results are very portable. You have probably changed something in the input, or are using a different pseudopotential, etc... You can also check earlier on that you are getting the same total energy forces etc...

The physical criterion is whether the end result is the same for the same required tolerances etc... The tests are not meant to be physically meaningful! They are meant to be quick and to port the same results on all platforms
