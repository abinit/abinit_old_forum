[Number of bands found less then required.
Calculation will proceed with nbnds = 200


nband specified in the input file is greater than the number of states
stored in the KSS file (200). You have to generate a new KSS file with more bands.

hdr_check: WARNING -
input ecut_eff= 4.998704 /= disk file ecut_eff= 16.000000.


You are truncating the planewave basis set used to describe the wavefunctions
in the GW part. ecutwfn is smaller that the value of ecut used during the KSS generation.

The last warning is not important

M
