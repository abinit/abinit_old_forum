Re: electron-phonon coupling with GGA+U

Posted: Mon May 26, 2014 10:53 pm
by mverstra
For the response theory, not yet. DFT+U requires PAW, and DFPT+U is in beta, and finally DFPT+GGA is not implemented. Few tests of DFPT+PAW+electron phonon coupling have been done so far, but with LDA much of it should work.

In frozen phonon you would have to calculate things yourself - for the diagonal elements of the coupling you can look at changes of the electron eigenvalues wrt phonon displacement, but for the full matrix of epc elements I don't know exactly how you could extract it. If you have a suggestion or a solution I am certainly interested to hear about it!

Matthieu
