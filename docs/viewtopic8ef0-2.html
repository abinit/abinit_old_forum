You can use eivec=2 to print the eigen vectors.

If you want a tool that directly reads the anaddb output you can use animemd in the abiout package : https://launchpad.net/~piti-diablotin/+ ... ntu/abiout
Open the file output with anaddb and switch to mode phonon with the comman

Code: Select all

:mode phonons
:add QX QY QZ IMODE

where QX QY QZ are one qpt and IMODE the mode number.
(see animemd --help)
